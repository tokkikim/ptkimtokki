# 레퍼런스 포즈 이미지 준비 가이드

이 폴더에는 포즈 생성에 사용할 레퍼런스 포즈 이미지들이 저장됩니다.

## 필요한 파일

다음 4개의 포즈 이미지 파일을 이 폴더에 저장하세요:

1. `rabbit-smile.png` - 웃는 포즈
2. `rabbit-point.png` - 손가락 포인트 포즈
3. `rabbit-think.png` - 생각하는 포즈
4. `rabbit-heart.png` - 하트 포즈

## 이미지 요구사항

- 형식: PNG (투명 배경 권장)
- 크기: 1024x1024 권장 (정사각형)
- 내용: 각 포즈를 보여주는 캐릭터 이미지 (토끼 등)

## 사용 방법

1. 위 4개 이미지 파일을 이 폴더(`public/reference-poses/`)에 저장
2. 사용자가 인물 이미지를 업로드하면
3. 서버가 각 레퍼런스 포즈를 참조하여 인물이 그 포즈를 취하도록 생성
4. 생성된 각 포즈에 배경 제거가 자동으로 적용됨

## 예시

- 레퍼런스: 토끼가 웃는 포즈
- 업로드: 사용자의 인물 사진
- 생성: 사용자가 웃는 포즈를 취한 이미지
